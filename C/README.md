# C

> NOTE: This folder is INCOMPLETE


> Notes, examples, and experiments from my journey learning C.

This directory contains the notes I took while learning the C programming language. Rather than writing long explanations, I prefer learning through code, so each topic is accompanied by runnable examples and detailed comments.

The goal is to create a collection of examples that are easy to revisit whenever I forget a concept—and hopefully useful for anyone else learning C.

---

## What you'll find

Each file focuses on a single topic and usually includes:

- 📖 Explanations in comments
- 💻 Runnable examples
- ⚠️ Common mistakes
- 💡 Helpful tips
- 📝 Notes about language behavior

---

## Learning Philosophy

I believe the best way to learn programming is to:

1. Learn a concept.
2. Experiment with it.
3. Build something using it.
4. Document what I learned.

These files are the result of that process.

---

## File Naming

Files are numbered in the approximate order I learned the topics.

Example:

```text
00-helloworld.c
01-variables.c
02-format-specifiers.c
...
```

The numbering makes it easier to follow along from the beginning or quickly find a topic later.

---

## Prerequisites

To compile the examples, you'll need a C compiler.

Using GCC:

```bash
gcc 01-variables.c -o variables
./variables
```

---

## Notes

- The examples are intentionally simple.
- Code readability is prioritized over clever tricks.
- Some examples may not represent production-quality code—they're meant to teach a concept.
